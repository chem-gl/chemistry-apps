Archivo: chemistry-apps:frontend/src/app/toxicity-properties/toxicity-properties.component.scss

Problema:
Unexpected duplicate selector ".metadata-grid p", first used at line 26

Razón:
Tipo: CODE_SMELL
Regla: css:S4666
Severidad: MAJOR
Línea: 171

Cómo solucionarlo:
To fix your code, either remove the duplicated selector or merge all declarations.

Noncompliant code example

p {
  color: blue;
  font-size: 16px;
}

p { /* Noncompliant: duplicated selector 'p', overwrites property 'color' */
  color: red;
}

Compliant solution

p {
  color: red;
  font-size: 16px;
}

Problema 2
Archivo: chemistry-apps:frontend/src/app/sa-score/sa-score.component.scss

Problema:
Text does not meet the minimal contrast requirement with its background.

Razón:
Tipo: CODE_SMELL
Regla: css:S7924
Severidad: MAJOR
Línea: 138

Cómo solucionarlo:
Increase the contrast by using darker text on light backgrounds or lighter text on dark backgrounds. Use online contrast checkers to verify your
color combinations meet WCAG standards. To check your contrast as you make your color choices, use a tool such as WebAIM’s Color Contrast Checker.

Noncompliant code example

.text {
  color: #777777; /* Light gray text */
  background-color: #ffffff; /* White background */
  /* Contrast ratio: 4.48:1 - fails WCAG AA for normal text */ // Noncompliant
}

Compliant solution

.text {
  color: #595959; /* Darker gray text */
  background-color: #ffffff; /* White background */
  /* Contrast ratio: 7.0:1 - passes WCAG AA */
}
