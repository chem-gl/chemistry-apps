# openapi

Este directorio contiene la definición OpenAPI de la API REST del backend. El archivo `openapi.yaml` es el contrato oficial que describe los endpoints, parámetros, respuestas y modelos de datos de la API. es autogenerado a partir de los decoradores y anotaciones en el código Python del backend, utilizando herramientas como FastAPI o Flask-RESTX. no se debe edutar manualmente
